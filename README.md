# E-Commerce Sales Analysis Dashboard

## Project Overview

This project analyzes e-commerce sales data using MySQL and Power BI to uncover business insights related to sales performance, profitability, customer behavior, and regional trends.

The project demonstrates skills in:

* SQL Data Analysis
* Data Cleaning
* Business Intelligence
* Dashboard Development
* Data Visualization
* DAX Measures

---

## Tools & Technologies

* MySQL
* Power BI
* DAX
* CSV Dataset

---

* Total Records: 9,994

Key fields include:

## Key Business Insights

* Technology is the highest-performing category.
* West region contributes the highest sales.
* Consumer segment generates the highest profit.
* California is the top-performing state.
* Profitability opportunities exist within the Furniture category.
